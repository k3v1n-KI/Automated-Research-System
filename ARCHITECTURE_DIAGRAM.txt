╚══════════════════════════════════════════════════════════════════════════════╝

SESSION 1: Initial Research
─────────────────────────────
Input:  prompt="Find hospitals in Ontario"
        no previous_session_id
        no tweak_instructions

Processing:
  Query Gen → LLM generates 4 queries
              ["Ontario hospitals", "Toronto clinics", ...]
  Search    → 50 URLs found
  Validate  → 45 URLs extracted
  Scrape    → 40 pages fetched
  Extract   → 87 records extracted (Name, Address, Website)
  Deduplicate → 87 unique records

Output: 
  • datasets/dataset_abc123.csv (87 records × 3 fields)
  • Vector Store: Session context saved
    - Queries: ["Ontario hospitals", ...]
    - Items: [{Name: "General Hospital", ...}, ...]
    - Metadata: {timestamp, count, fields}

SESSION 2: Refinement with Tweaks
──────────────────────────────────
Input:  prompt="Find hospitals in Ontario"
        previous_session_id="abc123..."
        tweaks="Also include phone numbers and operating hours"

Processing:
  Load Context → From vector store:
                 - Previous queries: ["Ontario hospitals", ...]
                 - Previous items: 87 records
  
  Query Gen → LLM sees previous queries + tweaks
              Generates 4 NEW queries
              ["Ontario hospital phone directory", ...]
              Combines: 4 old + 4 new = 8 total queries
  
  Search    → 8 queries × more comprehensive coverage
              75 URLs found (some overlap, some new)
  
  Validate  → 65 URLs extracted
  
  Scrape    → 55 pages fetched
  
  Extract   → 120 records extracted
              - 87 from previous queries (updated with phone/hours)
              - 33 new records from new queries
              Fields: Name, Address, Website, Phone, Hours
  
  Deduplicate → 120 unique records total

Output:
  • datasets/dataset_xyz789.csv (120 records × 5 fields)
  • All 87 previous records preserved
  • All 33 new records added
  • Field normalization: all records have all 5 fields
  • Vector Store: New session saved
    - Queries: 8 (4 old + 4 new)
    - Items: 120 records
    - Metadata: {linked_to_session_abc123, timestamp, ...}

SESSION 3: Third Iteration
──────────────────────────
Input:  prompt="Find hospitals in Ontario"
        previous_session_id="xyz789..." (continuing from Session 2)
        tweaks="Focus on rural areas, add director names"

Output:
  • 156 records total (120 previous + 36 new)
  • New fields: Name, Address, Website, Phone, Hours, Director
  • All previous records preserved and updated

┌─ context_persistence.py ────────────────────────────────────────────────────┐
│ ResearchContextManager                                                       │
│  • save_research_session(session_data) → saves to vector store              │
│  • load_session_context(session_id) → retrieves context                    │
│  • get_previous_queries(session_id) → get queries from past                 │
│  • get_previous_extracted_items(session_id) → get previous data            │
│  • analyze_data_gaps(session_id, instructions) → find missing info          │
│                                                                              │
│ SessionHistory                                                               │
│  • record_session(session_id, metadata)                                     │
│  • get_all_sessions() → list for dropdown                                   │
│  • get_latest_session() → most recent                                       │
└──────────────────────────────────────────────────────────────────────────────┘

┌─ algorithm.py (ResearchState TypedDict) ───────────────────────────────────┐
│ Fields (original):                                                           │
│  • initial_prompt, column_specs, queries, search_results, validated_urls   │
│  • scraped_content, extracted_items, final_dataset, session_id, round      │
│                                                                              │
│ Fields (NEW):                                                                │
│  • previous_session_id: Optional[str]  ← which session to continue from     │
│  • tweak_instructions: Optional[str]   ← what to refine                    │
│  • previous_queries: List[str]         ← from prior session                 │
│  • previous_items: List[Dict]          ← extracted data to build on         │
└──────────────────────────────────────────────────────────────────────────────┘

┌─ server.py (WebSocket Handlers) ────────────────────────────────────────────┐
│ @socketio.on('start_research')                                              │
│  • Accepts: prompt, previous_session_id (optional), tweaks (optional)       │
│  • Loads context if continuing from previous session                        │
│  • Creates state with all fields populated                                  │
│  • Invokes LangGraph with full state                                        │
│  • Saves results to vector store                                            │
│                                                                              │
│ @socketio.on('get_previous_sessions')                                       │
│  • Returns list of all previous sessions with timestamps                    │
│  • Populates dropdown on client side                                        │
│                                                                              │
│ handle_export_dataset()                                                      │
│  • Saves with field normalization                                           │
│  • All records have all fields                                              │
└──────────────────────────────────────────────────────────────────────────────┘
